Detailed Description

Classe Camion qui contient les informations sur un Camion Contient la plaque d'immatriculation, la date d'enregistrement, le poid, le nombre d'essieu et le NIV du camion. Des méthode qui permettent de nous donner chaques valeurs. Une méthode qui permet d'assigner une nouvelle plaque de camion. Une méthode qui permet d'afficher le tout dans un format console.
