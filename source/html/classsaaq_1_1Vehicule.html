Detailed Description

Classe vehicule qui contient les informations d'une plaque automobile. Contient la plaque d'immatriculation, la date d'enregistrement et le NIV du vehicule. Des méthode qui permettent de nous donner chaques valeurs. Une méthode qui permet d'assigner une nouvelle plaque de vehicule. Une méthode qui permet d'afficher le tout dans un format console.
